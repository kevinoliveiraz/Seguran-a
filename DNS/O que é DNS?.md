Toda vez que você digita um site, seu dispositivo pergunta a um servidor DNS: *"qual é o IP desse endereço?"* Por padrão, essa consulta vai para o servidor do seu provedor de internet — que registra tudo que você acessa, em texto claro, sem criptografia.

**O problema:** Mesmo com VPN ou Tor, um DNS mal configurado pode vazar seus sites visitados para o provedor. Isso se chama **DNS leak**.
